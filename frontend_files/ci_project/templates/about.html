Our Story

This is the live site for a software development course team project. It is built primarily with Django and MySQL database. Other aspects of the project included working on a 5-person team with Github as our project repository, as well as utilizing Agile methodologies with 2-week sprints followed by product demonstrations. In this course, students are divided into teams with similar interests, each team has one scrum master/product manager and the remaining members of the team function as engineers/developers During the sprints, roles are rotated and new tasks are assigned.
